## はじめに・使い方の注意点

本プロンプト集は「AI 論文読み方 やり方」に関する作業をAIで効率化するための実戦テンプレートです。

### ご利用前に必ずお読みください

1. **動作環境**：ChatGPT（GPT-4以上推奨）、Claude 3以上、Gemini 1.5以上で動作確認済み
2. **コピペで使えます**：プロンプト本文はそのままコピーして貼り付けてください
3. **カスタマイズ推奨**：`{変数名}` で囲まれた部分はご自身の状況に合わせて書き換えてください

5. **バージョン更新**：AIモデルのアップデートにより出力の質が変わる場合があります

## プロンプト集

はい、承知いたしました。AI論文の読み方に関するブログ記事のテーマに基づいた、実戦的なプロンプト3選を作成します。

---

### プロンプト1：基本編（初心者向け）
**タイトル**：AI論文の「超」要約！
**用途**：難解なAI論文の主要なポイントを短時間で把握する。
**プロンプト本文**：
```
あなたは、AI論文の専門家です。以下のAI論文の全文を読み、以下の指示に従って要約してください。

論文のタイトル: [論文のタイトルをここに貼り付ける]
論文のURL（もしあれば）: [論文のURLをここに貼り付ける]

指示:
1. この論文の最も重要な発見や貢献を3つ、箇条書きで挙げてください。
2. この論文の主な目的と、それがAI分野にどのような影響を与える可能性があるかを説明してください。
3. 専門用語を避け、AIに詳しくない人でも理解できるように、平易な言葉で説明してください。
4. 全体の要約は、150字以内にまとめてください。
```
**使い方のコツ**：論文のタイトルとURL（またはテキスト全体）を正確に貼り付けることで、AIが対象を特定しやすくなります。

---

### プロンプト2：応用編（中級者向け）
**タイトル**：AI論文の〇〇に特化した解説＆比較
**用途**：特定の技術や概念について、論文の解説と他の研究との比較を行う。
**プロンプト本文**：
```
あなたは、AI分野の最新研究動向を熟知したリサーチャーです。以下のAI論文を読み込み、指定されたトピックに焦点を当てて解説してください。

論文のタイトル: [論文のタイトルをここに貼り付ける]
論文のURL（もしあれば）: [論文のURLをここに貼り付ける]
解説してほしいトピック: {解説したい技術や概念、例:「提案手法の〇〇」「精度向上における△△の役割」}

指示:
1. 指定されたトピックについて、論文内でどのように説明されているかを具体的に解説してください。
2. そのトピックに関連する、過去の代表的な研究や、この論文が先行研究と比べてどのような進歩をもたらしているかを、もし可能であれば言及してください。
3. このトピックに関する専門用語は、簡単な注釈を加えてください。
4. 全体の解説は、300字〜500字程度にまとめてください。
```
**使い方のコツ**：`{解説したい技術や概念}`の部分を、論文を読み進める上で特に知りたい点や、他の論文との比較で重要になりそうな部分に具体的に置き換えてください。

---

### プロンプト3：上級テクニック編
**タイトル**：AI論文の「実装」に役立つ情報抽出＆質問生成
**用途**：論文で提案された手法の実装や検証に役立つ情報を抽出し、さらに深掘りするための質問を生成する。
**プロンプト本文**：
```
あなたは、AI論文の実装経験が豊富なエンジニアであり、鋭い質問を投げかける研究者でもあります。以下のAI論文を読み込み、実装とさらなる研究の観点から、最も重要な情報を抽出し、具体的な質問を生成してください。

論文のタイトル: [論文のタイトルをここに貼り付ける]
論文のURL（もしあれば）: [論文のURLをここに貼り付ける]

指示:
1. 論文で提案されている手法を実装する際に、特に注意すべき点や、鍵となるパラメータ、アルゴリズムのステップを箇条書きで抽出してください。
2. 論文の実験設定（データセット、評価指標、ベースラインモデルなど）で、実装や再現性を妨げる可能性のある曖昧な点や、さらなる情報が必要な点を特定してください。
3. この論文の研究をさらに発展させるために、どのような疑問点や未解決の問題があるかを3つ、質問形式で生成してください。これらの質問は、具体的な研究の方向性を示唆するものであるべきです。
4. 全体を通して、技術的な正確性を保ちつつ、実践的な視点からの情報提供を心がけてください。
```
**使い方のコツ**：このプロンプトは、論文の内容を深く理解し、実際に手を動かして検証したい場合に非常に役立ちます。抽出された情報や生成された質問を元に、さらにAIに質問を投げかけることで、より深い洞察を得られます。
