## はじめに・使い方の注意点

本プロンプト集は「AI Python コード 生成 テンプレ」に関する作業をAIで効率化するための実戦テンプレートです。

### ご利用前に必ずお読みください

1. **動作環境**：ChatGPT（GPT-4以上推奨）、Claude 3以上、Gemini 1.5以上で動作確認済み
2. **コピペで使えます**：プロンプト本文はそのままコピーして貼り付けてください
3. **カスタマイズ推奨**：`{変数名}` で囲まれた部分はご自身の状況に合わせて書き換えてください
4. **AIの出力は必ず確認を**：AIの回答は参考情報です。重要な判断はご自身で確認してください
5. **バージョン更新**：AIモデルのアップデートにより出力の質が変わる場合があります

## プロンプト集

はい、承知いたしました。AIプロンプトエンジニアリングの専門家として、ブログ記事のテーマとキーワードに基づいた「実戦プロンプト3選」を、指定された形式で作成します。

---

### プロンプト1：基本編（初心者向け）
**タイトル**：AIに「Pythonコード生成」をおねだりするプロンプト
**用途**：簡単なPythonコードをAIに生成してもらう
**プロンプト本文**：
```
あなたは優秀なPythonプログラマーです。
以下の要望に基づいて、Pythonコードを生成してください。

要望：
ユーザーからの入力を受け取り、その入力された文字列を画面に表示するPythonコードを作成してください。
```
**使い方のコツ**：具体的な要望を明確に伝えることが、AIが意図したコードを生成する鍵です。

---

### プロンプト2：応用編（中級者向け）
**タイトル**：Pythonコード生成の「テンプレート化」プロンプト
**用途**：様々な条件でPythonコードを生成できる汎用的なテンプレートを作成する
**プロンプト本文**：
```
あなたはPythonコード生成のエキスパートです。
以下のテンプレートを元に、{言語} で {機能} を実現するコードを生成してください。

テンプレート：
あなたは {役割} です。
{言語} を使って、以下の {機能} を実現するコードを生成してください。
コードには {ライブラリ} を使用してください。
コードの最後に、簡単な実行例と説明を加えてください。

{言語} = Python
{機能} = リスト内の数値の合計を計算する
{ライブラリ} = 基本機能のみ
{役割} = Pythonプログラマー
```
**使い方のコツ**：`{}` で囲まれた部分を、実現したい機能に合わせて具体的に書き換えることで、様々なコード生成に活用できます。

---

### プロンプト3：上級テクニック編
**タイトル**：AIと「対話しながらコードを改善」するプロンプト
**用途**：AIと対話しながら、生成されたPythonコードを段階的に改善していく
**プロンプト本文**：
```
あなたは優秀なPythonプログラマーであり、コードレビューの専門家です。
私が提示するPythonコードについて、以下の観点からレビューし、改善点を提案してください。

1.  **可読性**: コードが読みやすいか？変数名やコメントは適切か？
2.  **効率性**: より高速に、または少ないメモリで実行できるか？
3.  **堅牢性**: エラーハンドリングは適切か？予期せぬ入力に対応できるか？
4.  **Pythonic**: Pythonらしい書き方になっているか？

まず、私が提示するコードをレビューしてください。その後、改善案を提示し、私からの質問に答えてください。

（ここに、改善したいPythonコードを貼り付けてください）
```
**使い方のコツ**：AIに具体的なレビュー観点を指定することで、より的確なフィードバックを得られます。コードを貼り付けた後、AIの提案に対してさらに質問を投げかけることで、より洗練されたコードへと導くことができます。
