### プロンプト1：基本編（初心者向け）
**タイトル**：【初心者向け】条件付き書式を自動化するExcelマクロ作成
**用途**：指定した範囲のセルの値に基づいて、条件付き書式（セルの色付け）を自動で適用するVBAコードを生成します。
**プロンプト本文**：
```
あなたはExcel VBAの専門家です。
以下の条件を満たすExcelマクロ（VBAコード）を作成してください。

**マクロの機能:**
指定したExcelシートの、指定したセル範囲内の値を確認し、もしセルの値が「10」以上であれば、そのセルの背景色を「赤」に変更してください。
マクロを実行した際に、ユーザーに確認なしに直接色を付けるようにしてください。

**具体的な指示:**
- 対象シートは「Sheet1」としてください。
- 対象セル範囲は「A1:C10」としてください。
- 条件は「セルの値 >= 10」です。
- 適用する書式は「背景色を赤」です。
- VBAコードのみを出力してください。解説や説明は不要です。
```
**使い方のコツ**：生成されたVBAコードをExcelのVBAエディターに貼り付け、実行することで、指定した範囲の条件付き書式が自動で適用されます。シート名やセル範囲は、必要に応じてプロンプト本文を編集して調整してください。

---

### プロンプト2：応用編（中級者向け）
**タイトル**：【応用編】複数の条件でセルの色を変更するExcelマクロ作成
**用途**：複数の条件に基づいて、指定したセル範囲のセルの背景色や文字色をカスタマイズできるVBAコードを生成します。
**プロンプト本文**：
```
あなたはExcel VBAのプログラミングに精通したアシスタントです。
以下の仕様に基づいたVBAコードを作成してください。

**マクロの機能:**
指定されたExcelシートの、指定されたセル範囲において、複数の条件に基づいてセルの書式（背景色、文字色）を変更します。

**カスタマイズ変数:**
- `{シート名}`: マクロを適用するシート名を指定します。
- `{セル範囲}`: 書式を変更する対象のセル範囲を指定します。
- `{条件1}`: 最初の条件（例: "> 100"）
- `{色1}`: 条件1を満たした場合に適用する背景色（例: "vbYellow"）
- `{条件2}`: 2番目の条件（例: "< 50"）
- `{色2}`: 条件2を満たした場合に適用する背景色（例: "vbGreen"）
- `{条件3}`: 3番目の条件（例: "= 0"）
- `{色3}`: 条件3を満たした場合に適用する背景色（例: "vbRed"）
- `{文字色}`: 上記いずれの条件にも当てはまらない場合に適用する文字色（例: "vbBlack"）

**例:**
`{シート名}` = "売上データ"
`{セル範囲}` = "B2:F20"
`{条件1}` = "> 5000"
`{色1}` = "vbCyan"
`{条件2}` = "< 1000"
`{色2}` = "vbMagenta"
`{条件3}` = "= 0"
`{色3}` = "vbGray"
`{文字色}` = "vbBlue"

VBAコードのみを出力してください。
```
**使い方のコツ**：`{}`で囲まれた変数を、ご自身のExcelファイルの使用状況に合わせて具体的に置き換えてください。条件や色を増減させたい場合は、生成されたコードを参考に、ご自身で追加・編集してください。

---

### プロンプト3：上級テクニック編
**タイトル**：【上級テクニック】ユーザー定義関数（UDF）としてマクロを呼び出す
**用途**：Excelの関数のように使えるユーザー定義関数（UDF）を作成し、その関数内で条件付き書式を適用するVBAコードを生成します。これにより、より柔軟なマクロ活用が可能になります。
**プロンプト本文**：
```
あなたはExcel VBAと関数開発の専門家です。
Excelのシート上で直接呼び出せるユーザー定義関数（UDF）を作成してください。
このUDFは、指定されたセル範囲内の値を確認し、特定の条件を満たす場合に、そのセルの背景色を変更する処理を行います。

**UDFの仕様:**
- 関数名: `ApplyConditionalColor`
- 引数:
    - `TargetRange` As Range: 書式を適用するセル範囲
    - `ConditionValue` As Double: 条件となる値（例: 10）
    - `Operator` As String: 比較演算子（例: ">=", "<=", "=", ">", "<"）
    - `ColorCode` As Long: 適用する背景色のVBAコード（例: vbRed, RGB(255, 0, 0)）
- 処理内容:
    - `TargetRange`内の各セルについて、`ConditionValue`と`Operator`で指定された条件を評価します。
    - 条件を満たす場合、そのセルの背景色を`ColorCode`で指定された色に変更します。
    - この関数は、直接的な値を返さず、書式適用のみを行います（Subプロシージャのように振る舞わせます）。
- 出力: UDFのVBAコードのみを出力してください。

**例:**
`=ApplyConditionalColor(A1:C10, 10, ">=", vbYellow)`
この関数をシート上でこのように使用できるようにしてください。
```
**使い方のコツ**：生成された`ApplyConditionalColor`関数をVBAエディターに登録した後、Excelシート上で `=ApplyConditionalColor(適用したいセル範囲, 条件値, 演算子, 色コード)` の形式で入力して使用します。これにより、Excelの標準関数のように、数式バーから直接マクロの実行と条件設定が可能になります。
